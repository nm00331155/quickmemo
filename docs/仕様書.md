# QuickMemo 仕様書（実機抽出版）

## 1. この文書の目的
- 本書は、端末にインストール済みの `com.quickmemo.app` から抽出した一次情報のみを根拠に再構成した仕様書です。
- 既存仕様の混入を避けるため、実機証跡（`adb` / `run-as` / `dumpsys` / DB抽出）を優先しています。

- 証跡取得日時: 2026-03-06（主に 21:48-21:52 JST）

## 9. 未確定事項と制約
- DataStoreの利用中端末値は主要型（bool/int32/string）を復元済みですが、クリーン初回起動時は `quickmemo_settings.preferences_pb` 自体が未生成で、初期値はコード上の遅延既定値評価に依存します。
- `dumpsys package` はResolver中心のため、Manifest全コンポーネント定義を完全列挙する用途には追加検証が必要です。
- UIレイアウト・操作仕様の詳細は、端末UI観察またはソース解析を併用しない限り確定できません。
